# AGENTS.md

Guidance for AI coding assistants working in this directory.

This directory contains old and experimental GUI code.

Do not treat code under `src/gui/sketches/` as current production UI. It may be
useful as historical context or inspiration for future features, such as file
browsing or workspace management, but new application features should not be
built here unless the task explicitly asks for it.

If a task concerns the current MaweJS UI, start from `src/gui/app/`,
`src/gui/editor/`, `src/gui/common/`, or `src/slatejs/` instead.
